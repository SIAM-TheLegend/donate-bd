Important FAQ

What is Document Object Model (DOM)?

This is an JavaScript version of HTML. In this case, the whole HTML is converted into object, which is called Document. The whole thing is called Document Object Model.

How do you select an element from the DOM?

Generally we can select an element of DOM in many ways. Some of the popular methods of selecting element from DOM is by using 'document.querryselector()', 'document.querryselectorAll()', 'document.getElementById()', 'document.getElementsByClassName()' etc.

What is Event Delegation in the context of DOM, and why it's useful?

Event delegation is a technique where you attach an event listener to a parent element, rather than to each individual child element. This is useful because it improves performance, especially when dealing with a large number of child elements. Adding or removing child elements doesn't require updating event listeners, as the parent's listener handles events bubbling up from its children.

How to you manipulate an element's attributes and styles using DOM?

You can manipulate an element's attributes and styles using the DOM by accessing the element's properties. For attributes, use the `getAttribute()` and `setAttribute()` methods. For styles, you can access the `style` property directly or use `classList` to manage CSS classes.
